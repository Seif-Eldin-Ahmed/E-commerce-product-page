# 🛒 Sneaker E-Commerce Product Page

A modern and fully responsive e-commerce product page designed with a clean and user-friendly interface. This project focuses on delivering a smooth user experience through interactive elements and a well-structured layout.

## 🚀 Features
- 📱 Fully Responsive Design (Mobile, Tablet, Desktop)
- 🖼️ Interactive Product Image Gallery
- 🔍 Modal Preview with Slider Navigation
- ➕ Add to Cart with Quantity Control
- 🎯 Clean & Modern UI/UX
- 🧩 Well-Organized and Maintainable Code

## 🧠 Overview
This project showcases a sneaker product page where users can browse product images, open a modal preview with navigation, adjust item quantity, and add products to the cart.

The layout is carefully crafted to adapt seamlessly across all screen sizes, ensuring a consistent and engaging experience on any device.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS)

Responsive design optimized for all devices with smooth interactions and modern styling.
